Detailed Description

ARM interrupt controller.

This class provides generic functionity for installing and servicing interrupts on ARM. Each ARM system should provide a ARM interrupt controller implementation which inherits from this class.

See also
http://www.state-machine.com/arm/Building_bare-metal_ARM_with_GNU.pdf
