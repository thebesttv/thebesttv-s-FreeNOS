Detailed Description

Broadcom (2835) I2C controller implementation.

This class implements access to the I2C controller used in the Broadcom 2835 System-on-Chip. The controller features a 16-byte FIFO for data reads and writes and allows the use of interrupts.

See also
BCM2835 ARM Peripherals.pdf
http://www.airspayce.com/mikem/bcm2835/
