Detailed Description

Unidirectional point-to-point channel using shared memory.

Implemented by using two separated memory pages. The data page is for the consumer in which it only reads the incoming data payloads. The producer writes payloads to the data page. The feedback page is written only by the consumer, where it stores the feedback information from its consumption, such as the total bytes read and status.
